# Zellij キーバインド一覧

`config.kdl` で `clear-defaults=true` のもと定義された全バインド。

## モード遷移 (Normal から)

| キー | 遷移先 |
| --- | --- |
| `Ctrl g` | locked |
| `Ctrl p` | pane |
| `Ctrl t` | tab |
| `Ctrl n` | resize |
| `Ctrl h` | move |
| `Ctrl s` | scroll |
| `Ctrl o` | session |
| `Ctrl b` | tmux |
| `Ctrl q` | Zellij 終了 |

各モードからは対応する `Ctrl ○` で normal に戻る。`esc` / `enter` も基本 normal 復帰。

## Normal で常時有効 (shared_except "locked")

| キー | 動作 |
| --- | --- |
| `Alt ←` / `Alt h` | 左ペイン or 前タブへ |
| `Alt →` / `Alt l` | 右ペイン or 次タブへ |
| `Alt ↑` / `Alt k` | 上ペインへ |
| `Alt ↓` / `Alt j` | 下ペインへ |
| `Alt n` | 新規ペイン |
| `Alt f` | フロートペイン切替 |
| `Alt i` | タブを左へ移動 |
| `Alt o` | タブを右へ移動 |
| `Alt +` / `Alt =` | リサイズ 拡大 |
| `Alt -` | リサイズ 縮小 |
| `Alt [` | 前のスワップレイアウト |
| `Alt ]` | 次のスワップレイアウト |
| `Alt p` | グループ内ペイン切替 |
| `Alt Shift p` | グループマーキング切替 |
| `Super d` | 右に分割 (Ghostty `cmd+d` パススルー) |
| `Super Shift d` | 下に分割 (`cmd+shift+d`) |

## Pane モード (`Ctrl p`)

| キー | 動作 |
| --- | --- |
| `h` / `j` / `k` / `l` / 矢印 | フォーカス移動 |
| `p` | 次ペインへ |
| `n` | 新規ペイン |
| `d` | 下に分割 |
| `r` | 右に分割 |
| `s` | スタック分割 |
| `x` | ペインを閉じる |
| `f` | フルスクリーン切替 |
| `z` | ペイン枠切替 |
| `w` | フロートペイン切替 |
| `e` | 埋込/フロート切替 |
| `i` | ペインをピン留め |
| `c` | ペイン名変更 |

## Tab モード (`Ctrl t`)

| キー | 動作 |
| --- | --- |
| `1`–`9` | 対応タブへ直接ジャンプ |
| `h` / `k` / `←` / `↑` | 前のタブ |
| `j` / `l` / `↓` / `→` | 次のタブ |
| `tab` | 直前のタブへ |
| `n` | 新規タブ |
| `x` | タブを閉じる |
| `c` | タブ名変更 |
| `s` | 同期入力の切替 |
| `b` | ペインを別タブへ分離 |
| `[` | ペインを左のタブへ |
| `]` | ペインを右のタブへ |

## Resize モード (`Ctrl n`)

| キー | 動作 |
| --- | --- |
| `h` / `j` / `k` / `l` / 矢印 | その方向に拡大 |
| `H` / `J` / `K` / `L` | その方向に縮小 |
| `+` / `=` | 拡大 |
| `-` | 縮小 |

## Move モード (`Ctrl h`)

| キー | 動作 |
| --- | --- |
| `h` / `j` / `k` / `l` / 矢印 | ペインをその方向へ移動 |
| `n` / `tab` | 次のポジションへ |
| `p` | 前のポジションへ |

## Scroll モード (`Ctrl s`)

| キー | 動作 |
| --- | --- |
| `j` / `↓` | 1 行下スクロール |
| `k` / `↑` | 1 行上スクロール |
| `l` / `→` / `Ctrl f` / `PageDown` | ページダウン |
| `h` / `←` / `Ctrl b` / `PageUp` | ページアップ |
| `d` | 半ページ下 |
| `u` | 半ページ上 |
| `Ctrl c` | 最下部へ戻り normal へ |
| `e` | スクロールバックを `$EDITOR` で編集 |
| `s` | 検索入力へ |

## Entersearch / Search モード

Entersearch:

| キー | 動作 |
| --- | --- |
| `enter` | search モードへ |
| `esc` / `Ctrl c` | scroll に戻る |

Search (スクロール系キーも使える):

| キー | 動作 |
| --- | --- |
| `n` | 下方向検索 |
| `p` | 上方向検索 |
| `c` | 大小区別切替 |
| `o` | 単語一致切替 |
| `w` | ラップ切替 |

## Session モード (`Ctrl o`)

| キー | 動作 |
| --- | --- |
| `a` | About プラグイン |
| `c` | Configuration プラグイン |
| `p` | Plugin Manager |
| `w` | Session Manager |
| `s` | Share プラグイン |
| `d` | セッションからデタッチ |

## Tmux モード (`Ctrl b`)

| キー | 動作 |
| --- | --- |
| `"` | 下に分割 |
| `%` | 右に分割 |
| `c` | 新規タブ |
| `h` / `j` / `k` / `l` / 矢印 | フォーカス移動 |
| `o` | 次ペインへ |
| `z` | フルスクリーン切替 |
| `n` | 次タブ |
| `p` | 前タブ |
| `,` | タブ名変更 |
| `space` | 次のスワップレイアウト |
| `[` | scroll モードへ |
| `x` | ペインを閉じる |
| `d` | デタッチ |
| `Ctrl b` | `Ctrl b` をそのまま送出して normal へ |

## Rename モード

| モード | キー | 動作 |
| --- | --- | --- |
| renametab | `esc` | リネーム取消して tab モードへ |
| renamepane | `esc` | リネーム取消して pane モードへ |
| 共通 | `Ctrl c` | normal へ |
| 共通 | `enter` | 確定して normal へ |
